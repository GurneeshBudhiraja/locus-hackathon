# API Endpoints

## Contractors

**GET** `/api/contractors` - Get all contractors

**GET** `/api/contractors/[id]` - Get a single contractor by ID

**POST** `/api/contractors` - Create a new contractor

**PUT** `/api/contractors/[id]` - Update a contractor

**DELETE** `/api/contractors/[id]` - Delete a contractor

**GET** `/api/contractors/search?githubLogin=xxx&walletAddress=xxx&repoName=xxx&role=xxx` - Search contractors by filters
